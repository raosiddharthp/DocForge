Technical Stack & Local Setup

Core Technology Stack

  • Orchestration: LangGraph (stateful multi-agent workflow)
  • LLM Backend: Ollama (local execution of open-source models)
  • Default Model: DeepSeek-Coder or Qwen2.5-Coder (code-focused)
  • RAG Framework: LlamaIndex (document indexing and retrieval)
  • Vector Store: Chroma (local persistent embeddings)
  • Frontend: Streamlit (interactive dashboard with upload and generation)
  • Style Enforcement: Custom StyleEnforcer agent with uploaded guide

Local Installation Guide

# 1. Install Ollama (macOS/Linux/Windows)
curl -fsSL https://ollama.com/install.sh | sh

# 2. Pull recommended model
ollama pull deepseek-coder:6.7b   # or qwen2.5-coder:7b

# 3. Clone the repo
git clone https://github.com/raosiddharthp/DocForge.git
cd DocForge

# 4. Install Python dependencies
pip install -r requirements.txt

# 5. Run the app
streamlit run app.py

Supported Input Formats

  • Markdown (.md) – preferred
  • PDF (.pdf)
  • Plain text (.txt)
  • Future: Google Docs export, Confluence pages

System Requirements

  • 8GB RAM minimum (16GB recommended for larger specs)
  • 10GB disk space (models + indexes)
  • macOS, Linux, or Windows (WSL2)

Privacy & Security

  • No internet connection required after model download
  • All data stays on your device
  • Open-source code for review and customization
  • No telemetry or logging by default
